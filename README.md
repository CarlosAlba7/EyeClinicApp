    Project Description
This project is based on a medical database for an eye clinic. The app allows for users to sign in as employees or patients to look at or change information accordingly. The landing page allows you to either log in or sign up for an account (sign up only applies for patients).

/*PATIENT (remove this heading before submission)*/
While signed in as a patient, you will be able to book a new appointment, view your current appointments, view the doctors employed, view and purchase from the shop, and edit your patient profile. The patient dashboard will provide you paths to each of these options along with alerts for your next and recent appointments.
